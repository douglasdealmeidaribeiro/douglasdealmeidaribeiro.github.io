# Douglas Ribeiro | Página inicial para GitHub Pages

Página inicial estática para apresentar a atuação acadêmica e tecnológica de Douglas Ribeiro no GitHub, com foco em educação financeira, matemática financeira, estatística, dados, tecnologia e tomada de decisão.

## Destaques

- Landing page responsiva em HTML, CSS e JavaScript puro.
- Seções de apresentação, áreas de atuação, projetos, jornada, tecnologias e contato.
- Destaque integrado para o canal no YouTube **Números que Decidem**.
- Estrutura pronta para publicação no GitHub Pages.

## Visualização local

Abra o arquivo `index.html` diretamente no navegador ou execute um servidor local simples:

```powershell
python -m http.server 8000
```

Depois acesse `http://localhost:8000`.

## Publicação no GitHub Pages

1. Envie os arquivos deste projeto para um repositório no GitHub, como `douglasdealmeidaribeiro.github.io`.
2. No repositório, acesse `Settings`.
3. Entre em `Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Escolha a branch principal e a pasta raiz do projeto.
6. Salve e aguarde a URL de publicação.
